如何用 SnapTweak 测试这个页面

  1. 确保已加载扩展（chrome://extensions → 加载 dist 文件夹）
  2. 点击工具栏的 SnapTweak 图标 → Start Selection
  3. 用 Element 模式点击上面那个灰色按钮，或用 Area 模式拖拽框选
  4. 拖动选区四角/四边的圆点调整范围，或拖动选区整体移动，满意后点 Confirm
  5. 在弹出的输入框里写："把这个按钮改成蓝色 #2563EB，加大 padding，加圆角和阴影"
  6. 点 Generate Prompt 复制提示词，或配置 API Key 后点 AI Auto-Fix 直接生成代码
